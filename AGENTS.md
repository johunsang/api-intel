# API Intelligence Engine

SaaS API 마켓플레이스를 분석하여 API 트렌드, 가격 정보, 경쟁사 데이터를 수집·분석·시각화하는 인텔리전스 플랫폼.

## 주의사항 
- 배포전에 

## 기술 스택

- **Framework**: Next.js 16 (App Router)
- **UI**: React 19, Tailwind CSS 4
- **상태관리**: Zustand 5
- **DB**: Supabase (PostgreSQL)
- **인증**: Supabase Auth
- **배포**: Vercel
- **패키지 매니저**: pnpm
- **모노레포**: Turborepo (`apps/web`)
